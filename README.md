# Sentiment Analysis Project

## Project Overview
This project analyzes Amazon product reviews and classifies them into Positive, Negative, and Neutral sentiments using Python.

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

## Dataset
Amazon Product Reviews Dataset (Kaggle)

## Results
- Positive Reviews: 1479
- Negative Reviews: 77
- Neutral Reviews: 41

## Features
- Reads Amazon review dataset
- Performs sentiment analysis
- Classifies reviews into Positive, Negative, and Neutral
- Displays sentiment counts
- Generates a bar graph
